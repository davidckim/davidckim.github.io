Ruby's ".each" For Javascript
11/02/2014

A long long time ago, in a galaxy far away when I was a young Jedi, I was told by seasoned programmers that it is a lot easier to pick up other languages once you learn one of them. They were right.. somewhat, only they forgot to mention that the terminologies could overlap but not mean the same thing. One thing I was pretty accustomed to in Ruby was utilizing the each method to iterate over element of arrays or data structures. In Ruby if you wanted to print out all the numbers within an array you would use :
[1, 2, 3, 4, 5].each { |element| puts "#{element}" }.
To do the same thing in JavaScript you would something like this:
var array = [1, 2, 3, 4, 5];
for (var i = 0; i < i.length; ++i) {
var element = a[i];
alert(i + ": " + element);
}

Thing like this make me appreciate the beauty of Ruby a lot more. Things in JavaScript seem so stretched out. Technically you could create a function that would do the same thing as each, but the simplicity of Ruby makes it a lot more appealing and attracts in a lot of soon-to-be programmers.
