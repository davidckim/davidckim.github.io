JavaScript Objects Cheatsheet

Object Creation

// Literal Notation
var objectName = {
  property1: value1,
  property2: value2
}

Accessing Object Properties

objectName.property1;    // => value1
objectName["property2"];  // => value2

Adding, Modifying, Deleting Object Properties

// Adding
objectName.property3 = value3;
objectName["property3"] = value3;

// Modifying
objectName.property1 = value4;
objectName.["property1"] = value4;

// Deleting
delete objectName.property3;
delete objectName["property3"];
